# برنامج التحول المؤسسي للنيابة العامة
## منصة متابعة المشروع

موقع متكامل لمتابعة أداء مشروع التشخيص المؤسسي وخارطة الطريق.

### الصفحات:
- `index.html` — الصفحة الرئيسية
- `dashboard.html` — لوحة متابعة الأداء (١٠٠ ساعة الأولى)
- `dashboard2.html` — لوحة مؤشرات المشروع
- `gantt.html` — المخطط الزمني

### تفعيل GitHub Pages:
1. Settings → Pages
2. Branch: main → / (root)
3. Save

- ١٢ أسبوع عمل
